O processo TDD utilizando testes funcionais e testes de unidade pode ser resumido a executar o fluxo de desenvolvimento aplicando TDD com testes funcionais e para cada feature descrita nesses testes é aplicado o processo de TDD utilizando os testes de unidade. Ou seja, gera-se o teste funcional para uma feature e a partir deste teste falhando gera-se testes de unidade para implementar incrementalmente à feature. Ao final do processo de TDD mais interno, executa-se os testes funcionais de novo, voltando ao TDD mais externo.
